# Felipe Neo4j Project

Este projeto é um estudo prático de **bancos de dados orientados a grafos** utilizando **Neo4j** e a linguagem de consulta **Cypher**. O objetivo principal é aprender a modelar dados em grafos, criar relacionamentos entre nós e realizar consultas avançadas para extrair informações significativas.

## Estrutura do Projeto

- `dataset/` : Contém os arquivos de dados utilizados para popular o grafo.
- `queries/` : Contém exemplos de consultas em Cypher para exploração do grafo.
- `images/` : Ilustrações e diagramas representando a estrutura do grafo.
- `cypher/` : Scripts utilizados para criar e manipular nós e relacionamentos no Neo4j.

## Funcionalidades Implementadas

- Criação de **nós** representando entidades, como pessoas, departamentos e produtos.
- Definição de **relacionamentos** entre os nós para modelar interações reais.
- Consultas usando **MATCH, WHERE, RETURN, COUNT e outras funções agregadas**.
- Exploração de caminhos e redes de relacionamentos entre entidades.

## Objetivo de Aprendizado

- Compreender o conceito de grafos e relacionamentos entre dados.
- Aplicar consultas complexas em Neo4j para análise de informações.
- Preparar um projeto estruturado que possa servir como portfólio para **análise de dados com grafos**.

## Tecnologias Utilizadas

- [Neo4j](https://neo4j.com/) – Banco de dados de grafos.
- Cypher – Linguagem de consulta do Neo4j.
- CSV – Para importação de datasets no grafo.
